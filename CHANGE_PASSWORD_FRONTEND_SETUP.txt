PROTRADE CHANGE PASSWORD FRONTEND UPDATE

This package adds a professional Security tab to settings.html.

Features:
- Current password field
- New password and confirmation
- Password show/hide controls
- Live strength requirements
- Calls PUT /api/auth/change-password
- Replaces the old browser JWT with the fresh token returned by the server
- Displays backend validation errors clearly
